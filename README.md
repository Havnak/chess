# chess
A chess tui in python

### Implemented rules:
- All movesets
- En passant
- Check
- Pinning to king
- King not allowed to move into check
- Not allowed to move, and still be in check
- Checkmate
- Stalemate
- Casteling
- 50 moves draw
- Repetition draw
- Promotion
